Tabs - lazy loading

Use tabs to define content sections and lazily load the content when the tab is selected, in the same way as for the accordion.

Demo

#{include 'jqueryui/Tabs/tabs.html'/}

Client

The widget is based on a a list of links to the Ajax content that will populate them.

#{includeVerbatim 'jqueryui/Tabs/tabs.html'/}

The tabs.js JavaScript applies the jQuery UI tabs plug-in to a list of links to the content. Note that this JavaScript has very little to do, because the tabs plug-in supports Ajax loading out of the box.

#{includeVerbatim '/public/javascripts/tabs.js'/}

Server

The tabs use the same jqueryui.Accordion.region(String header) action and jqueryui/Accordion/region.html template as the accordion.
