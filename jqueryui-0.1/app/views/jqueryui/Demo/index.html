jQuery UI module demo

accordion - lazy loading

Use an accordion to define content sections and lazily load the content.

Africa

America

Asia

Australia

Europe

Client

The accordion.js JavaScript applies the jQuery UI accordion plug-in to a DIV that contains headings, using the data-url attribute value for the server-side data source. Note the empty action method parameter value, which generates the query string part of the URL.

<div class="accordion" data-url="@{jqueryui.Accordion.region('')}">
	<h3><a href="#">Africa</a></h3>
	<h3><a href="#">America</a></h3>
	<h3><a href="#">Asia</a></h3>
	<h3><a href="#">Australia</a></h3>
	<h3><a href="#">Europe</a></h3>
</div>

Server

The jqueryui.Accordion(String header) action renders an HTML content section for the clicked header.

autocomplete - label only

Use a label-only autocomplete to suggest text values for an input field.

Type to populate the list, and select a label from the list.

Client

The autocomplete.js JavaScript applies the jQuery UI autocomplete plug-in to the following label, using the data-url attribute value for the server-side data source.

<input name="location" class="autocomplete" data-url="@{jqueryui.Autocomplete.label()}">

Server

The jqueryui.Autocomplete.label(String term) action renders a JSON-encoded list of strings that match the user-entered term parameter.

autocomplete - label and value

Use a label/value autocomplete to suggest input options that have a separate label and value.

Type to populate the list, then select the value for a particular label.

Submit the form to submit the numeric value, but not the label.

Client

This client implementation uses more JavaScript in autocomplete.js to handle label/value pairs, with an additional hidden field in the form.

Server

The jqueryui.Autocomplete.value(String term) action renders a JSON-encoded list of label/value pairs that match the user-entered term parameter.
