jQuery UI module demo

autocomplete - label only

Type to populate the list, and select a label from the list.

Client

The jqueryui-client.js JavaScript applies the jQuery UI autocomplete plugin to the following label, usign the data-url attribute value for the server-side data source.

<input name="location" class="autocomplete"
		data-url="@{jqueryui.Demo.autocompleteLabel()}">

Server

The jqueryui.Demo.autocompleteLabel(String term) action renders a JSON-encoded list of strings that match the user-entered term parameter.

autocomplete - label and value

Type to populate the list, then select the value for a particular label.

Submit the form to submit the numeric value, but not the label.

Client

This client implementation adds more JavaScript in jqueryui-client.js to handle label/value pairs, with an additional hidden field in the form.

Server

The jqueryui.Demo.autocompleteLabelAndValue(String term) action renders a JSON-encoded list of label/value pairs that match the user-entered term parameter.
