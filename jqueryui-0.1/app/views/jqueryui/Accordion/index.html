Accordion - lazy loading

Use an accordion to define content sections and lazily load the content when the section is opened.

Client

The accordion.js JavaScript applies the jQuery UI accordion plug-in to a DIV that contains headings, using the data-url attribute value for the server-side data source. Note the empty action method parameter value, which generates the query string part of the URL. Also, it would be straightforward to modify accordion.js to use a data-url attribute on each h3 element to specify the URL for each section.

<div class="accordion" data-url="@{jqueryui.Accordion.region('')}">
<h3><a href="#">Africa</a></h3>
<h3><a href="#">America</a></h3>
<h3><a href="#">Asia</a></h3>
<h3><a href="#">Australia</a></h3>
<h3><a href="#">Europe</a></h3>
</div>

Server

The jqueryui.Accordion.region(String header) action renders an HTML content section for the clicked header.
